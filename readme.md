# Seattle Crime Pulse (2025–2026)

## Overview

This project is an interactive web-based dashboard visualizing crime patterns across Seattle neighborhoods for 2025–2026.

The dashboard combines:
- A choropleth map of total crimes by neighborhood
- A dynamic popup showing crime totals
- A summary panel displaying total crime count
- A bar chart showing the Top 5 neighborhoods by crime

The goal of this project is to explore spatial patterns of crime and highlight neighborhoods with the highest reported activity.

---

## Data Sources

- Seattle Police Department (SPD) Crime Data (2025–2026)
- Seattle Neighborhood Boundaries (GeoJSON)

Processed files:
- `spd_crime_2025_2026_clean.csv`
- `crime_by_neighborhood_matched.csv`
- `seattle_neighborhoods.geojson`
